# 📝 To-Do List (CLI-based)

A simple command-line To-Do List application written in Python.  
Users can add, view, complete, and delete tasks efficiently.

## 🚀 Features
- ✅ Add new tasks
- 📜 View all tasks
- ✔️ Mark tasks as complete
- 🗑️ Delete tasks
